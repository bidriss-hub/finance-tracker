# Finance Tracker

## Features
- Add / Edit / Delete transactions
- Regex validation (4 rules)
- Search with regex + highlight
- Sorting (date, amount, category)
- LocalStorage persistence


## Keyboard
- Tab navigation supported
- Enter submits form

## Setup
Open index.html in browser
